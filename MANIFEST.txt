USAGE
=====

Quick Start:
1. docker compose up -d
2. Access frontend at http://localhost:3010
3. Backend API at http://localhost:8010
4. Embedding service at http://localhost:8011

Initialize Database:
1. docker exec -i postgres psql -U postgres -d appdb < backend/schema.sql
2. docker run --rm --network postgres-stack_default ... python seed.py

Verify Setup:
1. curl http://localhost:8010/api/health
2. curl http://localhost:8011/health
3. curl http://localhost:3010

API Examples:
- Search: GET /api/sections/search?q=fire+safety
- Chat: POST /api/chat with {"message": "..."}
- Embed: POST /api/embed with {"text": "..."}
